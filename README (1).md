# Bharat Innovation Challenge 2.0 — Website

A single-page website built from the BIC 2.0 Student Handbook (Lovely Professional University).

## Contents
- `index.html` — the entire site (HTML + CSS + JS in one file, no build step)
- `.nojekyll` — tells GitHub Pages to serve the files as-is

## Run locally
Just open `index.html` in a browser. That's it.

## Deploy on GitHub Pages
1. Push these files to the root of a GitHub repository.
2. Repo → **Settings** → **Pages**.
3. Under **Build and deployment**, set Source = **Deploy from a branch**.
4. Branch = `main`, folder = `/ (root)` → **Save**.
5. Wait ~1 minute. The live URL appears at the top of the same page:
   `https://<your-username>.github.io/<repo-name>/`

## Notes
- Fonts load from Google Fonts, so the page needs an internet connection to look right.
- Content is from the official BIC 2.0 handbook; verify dates, prize amounts and contact
  details against LPU's official page before sharing publicly.
